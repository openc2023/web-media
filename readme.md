## 📖 项目简介

这是一个收集平时自己做的小东西的项目集合。包含了各种Web技术实验、创意项目和学习成果。通过这些小项目，我不断学习新的技术和工具，提升自己的编程能力和创意表达能力。

## 📁 项目结构

```
web-media/
├── inno-things/          # 创新项目目录
│   ├── Top/              # 精选项目
│   └── index.html        # 创新项目主页
├── shanhaiart/           # 山海艺术项目
├── index.html            # 项目集合主页
└── readme.md             # 项目说明
```

## 🚀 快速开始

1. **克隆仓库**
   ```bash
   git clone https://github.com/openc2023/web-media.git
   ```

2. **启动服务器**
   ```bash
   # 使用 http-server
   npx http-server
   ```

3. **访问项目**
   ```
   http://localhost:8080
   ```
