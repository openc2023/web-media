# 个人项目集合

这是一个收集平时自己做的小东西的项目集合。包含了各种Web技术实验、创意项目和学习成果。

## 项目结构

```
web-media/
├── inno-things/          # 创新项目目录
│   ├── Top/              # 精选项目
│   └── index.html        # 创新项目主页
├── shanhaiart/           # 山海艺术项目
│   ├── old/              # 旧版本文件
│   ├── shanhai/          # 当前版本
│   ├── test/             # 测试文件
│   ├── .gitignore
│   ├── index.html
│   └── readme.md
├── index.html            # 项目集合主页
└── readme.md             # 项目说明
```

## 项目分类

- **山海艺术项目**：基于Web技术的新媒体艺术交互项目，通过HTML5 Canvas创建动态视觉效果
- **创新项目**：各种创新的Web实验和创意项目

## 如何使用

1. 克隆仓库
   ```bash
   git clone https://github.com/openc2023/web-media.git
   ```

2. 使用Web服务器打开项目
   ```bash
   # 如果安装了Node.js，可以使用http-server
   npx http-server
   ```

3. 在浏览器中访问
   ```
   http://localhost:8080
   ```

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
- Canvas API
