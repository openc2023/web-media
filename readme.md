## 📖 项目简介

这是一个收集平时自己做的小东西的项目集合。包含了各种Web技术实验、创意项目和学习成果。通过这些小项目，我不断学习新的技术和工具，提升自己的编程能力和创意表达能力。

## 🛠️ 技术栈

| 类别 | 技术 |
|------|------|
| 前端基础 | HTML5, CSS3, JavaScript (原生) |
| 图形技术 | Canvas API |
| 开发工具 | Git, VS Code |

## 🎯 项目特点

- **创意实验**：探索各种Web技术的可能性
- **交互体验**：注重用户交互和视觉效果
- **代码质量**：保持简洁清晰的代码风格
- **持续更新**：不断添加新的项目和功能

## 📁 项目分类

- **山海艺术项目**：基于Canvas的动态视觉效果
- **创新项目**：各种Web技术实验和创意作品
- **精选项目**：经过精心打磨的优质作品

## 💻 开发环境

```bash
# 克隆仓库
git clone https://github.com/openc2023/web-media.git

# 进入项目目录
cd web-media

# 启动本地服务器（推荐使用）
npx http-server
```
