Test Steps

统一测试步骤

  1. 用同一台手机，同一浏览器，同一 GitHub Pages 链接测试。
  2. 固定光线，先从 40-60cm 距离对准整张 000-top。
  3. 记录首次识别耗时。
  4. 识别后慢慢上下左右移动手机，观察模型是否抖动。
  5. 移出画作再移回来，记录恢复速度。
  6. 检查 box.glb 是否贴在同一个画作位置。
